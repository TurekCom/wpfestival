# Legal note

This repository publishes the source tree openly and keeps the original reverse-engineering work visible:

- original wrapper code written during the porting work
- packaging scripts
- documentation
- upstream Festival / Speech Tools sources under their original licenses
- documentation and build scripts

The git history does not publish:

- original WP installer binaries
- recovered WP voice payloads
- extracted `wp_runtime_lib`
- release signing keys

GitHub releases may include convenience binaries for end users in addition to source archives. If you mirror or redistribute those releases elsewhere, review the rights around the historical WP runtime materials yourself. If you have rights to the original WP assets, place them locally in the paths described in [BUILDING.md](BUILDING.md).
